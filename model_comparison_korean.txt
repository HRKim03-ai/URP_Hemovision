Fusion Phase 1 모델 선택: w_demo vs wo_demo 객관적 비교 자료
================================================================================

본 문서는 Fusion Phase 1 실험 결과를 바탕으로 두 가지 모델 버전의 성능을 
객관적으로 비교 분석한 자료입니다. 모델 선택 시 참고하시기 바랍니다.

--------------------------------------------------------------------------------
1. 실험 개요
--------------------------------------------------------------------------------

- w_demo: 메타데이터(나이, 성별) 포함 모델
- wo_demo: 메타데이터 제외 모델
- 평가 지표: MAE, R², Acc@0.5, Acc@1.0, Acc@2.0
- 비교 기준: R² > 0인 모델 중 MAE가 가장 작은 모델을 최고 성능 모델로 선정

4. 상세 비교 분석
--------------------------------------------------------------------------------

4.1 w_demo (메타데이터 포함)의 특징

장점:
  ✓ 최고 성능 모델의 MAE가 더 낮음 (0.2023 vs 0.2048)
  ✓ 최고 성능 모델의 Acc@0.5가 완벽함 (1.0000 vs 0.9375)
  ✓ 최고 성능 모델의 모든 Accuracy 지표가 완벽함
  ✓ 평균 Acc@1.0과 Acc@2.0이 더 높음
  ✓ R² > 0인 모델이 더 많음 (20개 vs 19개)

단점:
  ✗ 평균 R²가 더 낮음 (0.1597 vs 0.1801)
  ✗ 평균 MAE가 더 높음 (0.8551 vs 0.8365)
  ✗ 최고 성능 모델의 R²가 현저히 낮음 (0.0268 vs 0.0904, 약 3.4배 차이)
  ✗ 배포 시 메타데이터(나이, 성별) 수집 필요
  ✗ 개인정보 보호 이슈 고려 필요

4.2 wo_demo (메타데이터 제외)의 특징

장점:
  ✓ 평균 R²가 더 높음 (0.1801 vs 0.1597) - 모델 설명력이 더 좋음
  ✓ 평균 MAE가 더 낮음 (0.8365 vs 0.8551) - 평균적으로 오차가 더 작음
  ✓ 최고 성능 모델의 R²가 훨씬 높음 (0.0904 vs 0.0268, 약 3.4배)
  ✓ 배포 시 메타데이터 수집 불필요 - 실용적
  ✓ 개인정보 보호 이슈 없음

단점:
  ✗ 최고 성능 모델의 MAE가 약간 높음 (0.2048 vs 0.2023)
  ✗ 최고 성능 모델의 Acc@0.5가 완벽하지 않음 (0.9375 vs 1.0000)
  ✗ 평균 Acc@1.0과 Acc@2.0이 약간 낮음

--------------------------------------------------------------------------------
5. 선택 기준 가이드
--------------------------------------------------------------------------------

다음과 같은 상황에서는 w_demo를 고려하세요:

  [ ] 최고 정확도가 중요한 경우 (Acc@0.5 = 1.0 필수)
  [ ] 최고 성능 모델의 성능이 평균보다 더 중요한 경우
  [ ] MAE가 최소화되어야 하는 경우
  [ ] 메타데이터 수집이 가능한 환경
  [ ] 개인정보 보호 이슈가 없는 경우
  [ ] 모든 Accuracy 지표가 완벽해야 하는 경우

다음과 같은 상황에서는 wo_demo를 고려하세요:

  [ ] 모델의 설명력(R²)이 중요한 경우
  [ ] 평균 성능이 중요한 경우
  [ ] 실용적인 배포가 중요한 경우 (메타데이터 수집 어려움)
  [ ] 개인정보 보호 규정이 엄격한 경우
  [ ] 배포 단순화가 필요한 경우
  [ ] 모델의 일반화 성능이 중요한 경우

--------------------------------------------------------------------------------
6. 추가 고려사항
--------------------------------------------------------------------------------

연구 목적:
  • 논문 작성 시 모델 설명력(R²)이 중요하다면 → wo_demo
  • 최고 성능을 강조해야 한다면 → w_demo

실용성:
  • 실제 배포 환경에서 메타데이터 수집이 어렵다면 → wo_demo
  • 메타데이터 수집이 용이하다면 → w_demo

정확도 요구사항:
  • Acc@0.5 = 1.0이 필수적이라면 → w_demo
  • 약간의 오차가 허용된다면 → wo_demo
