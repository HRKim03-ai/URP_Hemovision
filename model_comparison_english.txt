
================================================================================
Fusion Phase 1 Model Selection: w_demo vs wo_demo Objective Comparison
================================================================================

This document provides an objective comparative analysis of two model versions 
based on Fusion Phase 1 experimental results. Please use this as a reference 
for model selection.

--------------------------------------------------------------------------------
1. Experiment Overview
--------------------------------------------------------------------------------

- w_demo: Model with demographic features (age, gender)
- wo_demo: Model without demographic features
- Evaluation metrics: MAE, R², Acc@0.5, Acc@1.0, Acc@2.0
- Comparison criterion: Best model is selected as the one with minimum MAE 
  among models with R² > 0

--------------------------------------------------------------------------------
4. Detailed Comparative Analysis
--------------------------------------------------------------------------------

4.1 Characteristics of w_demo (with demographic features)

Advantages:
  ✓ Lower MAE in best model (0.2023 vs 0.2048)
  ✓ Perfect Acc@0.5 in best model (1.0000 vs 0.9375)
  ✓ All accuracy metrics are perfect in best model
  ✓ Higher mean Acc@1.0 and Acc@2.0
  ✓ More models with R² > 0 (20 vs 19)

Disadvantages:
  ✗ Lower mean R² (0.1597 vs 0.1801)
  ✗ Higher mean MAE (0.8551 vs 0.8365)
  ✗ Significantly lower R² in best model (0.0268 vs 0.0904, approximately 3.4x difference)
  ✗ Requires demographic data (age, gender) collection for deployment
  ✗ Privacy concerns need to be considered

4.2 Characteristics of wo_demo (without demographic features)

Advantages:
  ✓ Higher mean R² (0.1801 vs 0.1597) - better model explanation
  ✓ Lower mean MAE (0.8365 vs 0.8551) - lower average error
  ✓ Much higher R² in best model (0.0904 vs 0.0268, approximately 3.4x)
  ✓ No need for demographic data collection - practical for deployment
  ✓ No privacy concerns

Disadvantages:
  ✗ Slightly higher MAE in best model (0.2048 vs 0.2023)
  ✗ Not perfect Acc@0.5 in best model (0.9375 vs 1.0000)
  ✗ Slightly lower mean Acc@1.0 and Acc@2.0

--------------------------------------------------------------------------------
5. Additional Considerations
--------------------------------------------------------------------------------

Research Purpose:
  • If model explanation (R²) is important for paper writing → wo_demo
  • If best performance needs to be emphasized → w_demo

Practicality:
  • If demographic data collection is difficult in deployment → wo_demo
  • If demographic data collection is feasible → w_demo

Accuracy Requirements:
  • If Acc@0.5 = 1.0 is essential → w_demo
  • If slight error is acceptable → wo_demo
